# Content Generation

## Python Environment Install 

### Create Python 3 virtual environment:
```sh
 python3 -m venv venv
```
### Activate virtual environment
```sh
 source venv/bin/activate
```
### Install dependencies 
```sh
 pip -r venv_req.txt
```
